Структура проекта:

  - папка parsing: содержит файл "main_parsep.py". Это парсер для обучающией модели, который берет 
    статьи из новостного сайта и записывает в csv файл.
    
  - папка preprocessing: содержит файл: "preoprocess.py". Этот файл предназначен для очистки
    и обработки файла, полученного в результате работы "main_parsep.py".

  - папка classification: содержит в себе уже готовую и обученую модель ("my_model.joblib"),
    готовый векторизатор ("vectorizer.joblib"), а также файл "classification.py", с помощью которого
    были получены вышеуказанные файлы.

  - файл "main.py": содержит в себе самого телеграм бота.



Для запуска проекта необходимо:

- Скачать файлы "main.py", "my_model.joblib" и "vectorizer.joblib".
- Положить их в одну директорию.
- Вставить токен вашего бота в TOKEN_API.
- Запустить проект.
